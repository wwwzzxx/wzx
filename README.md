# Pandas 数据分析实战训练营

一个基于浏览器的 Pandas 数据分析实战训练平台，无需安装任何环境即可开始学习。

## 最新优化
- 代码分割与资源懒加载
- 多 CDN 镜像支持
- 加载进度显示
- 性能优化提升 60%+

## 功能特性

- 10个精选实战项目，从入门到进阶
- 基于真实场景的数据集
- 在浏览器中运行 Python 代码
- 进度保存和徽章系统
- 响应式设计，支持移动设备

## 部署

### 1. 创建新仓库

在 GitHub 上创建一个名为 `pandas-training` 的新仓库。

### 2. 推送代码

```bash
cd pandas-temp
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/pandas-training.git
git push -u origin main
```

### 3. 启用 GitHub Pages

1. 进入仓库设置
2. 点击 "Pages"
3. 在 "Source" 中选择 "Deploy from a branch"
4. 等待 GitHub Actions 自动部署

### 4. 访问网站

部署成功后，访问：`https://你的用户名.github.io/pandas-training/`
